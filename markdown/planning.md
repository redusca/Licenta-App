# File orchestrator ->
App that will work with your file system for the best experience that you can have.

# The project splits in 3 main parts : 
## **1.The server** :
- Account managing
- Extensions - user made components for the app that will be send as extensions to be dowlanded in the app
- Contrainer deployer

## **2.The container**:
- Is having all the agent logic inside that can be deployed based on user request : 
    1. On the server of the app (limited to the plan you use)
    2. On personal Computer / Service you want. 
        - you will get the container that you can deploy anywhere you want.
- This container will do a secure tunnel conection to your app where you can you the agent for task and the agent can use tools from the app at user's will.

## **3. The app**:
- The main idea of the project, a place where users can work with files by the click of a button.
- Main feature : virtual drives - you can use multiple filters to move or create shortcuts to the files in your system without needing you to do that all the work manually
- Tools : tools that are build in to work with different type of files. For example : " Create a drive with images sorted by an AI (**requires containeir**) " , or audio file converter or ... A list of tools to be made will be written.
- Agent chat : can you yours tools but do more based on the request. HE can be used as a personal assitant in managing your files. 
 - Extensions : People can make extension for this project so it can add more tool , feature to the drive or customization to the app. ( 3 - 4 extension will be made for this project)
